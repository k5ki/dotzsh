# dotzsh

zsh config sharing for macOS users.

# 付録

## チートシート

### シェル

| キー   | アクション                       |
| ------ | -------------------------------- |
| Ctrl+A | 行頭に移動                       |
| Ctrl+E | 行末に移動                       |
| Ctrl+F | →                                |
| Ctrl+B | ←                                |
| Ctrl+P | ↑                                |
| Ctrl+N | ↓                                |
| Ctrl+K | カーソル位置から行末までを削除   |
| Ctrl+W | カーソル位置後ろの単語を一つ削除 |
| Ctrl+R | 履歴検索                         |
| Meta+F | 一単語分 →                       |
| Meta+B | 一単語分 ←                       |

### Ghostty

| キー        | アクション     |
| ----------- | -------------- |
| Cmd+Shift+, | 設定の反映     |
| Cmd+T       | タブを開く     |
| Cmd+W       | タブを閉じる   |
| Cmd+Shift+] | 右のタブに移動 |
| Cmd+Shift+[ | 左のタブに移動 |
